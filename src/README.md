### Baxe backend challenge.
##### To start, execute:
- docker-compose up -d (docker-compose down --rmi local -> To stop and remove built containers)
- ts-node src/index.ts (Ctrl+C -> To stop)

#### To run tests, execute:
- docker-compose up -d (docker-compose down --rmi local -> To stop and remove built containers)
- npm run test:watch (Ctrl+C -> To stop)
